# 基本认识
- 机器学习不是解决安全问题的银弹
- 安全的黑样本往往不足，如何在小样本量的基础上优化算法需要特别注意
- 模型的误杀率往往比较高，需要在保证正确率的基础上优化召回率
- 在鉴黑的同时要注意鉴白
